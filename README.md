# CollabDocs API

## 📌 Project Overview
CollabDocs is a backend API system similar to Notion or Google Docs where users can:
- Create workspaces
- Add members with roles
- Create and manage documents
- Track document versions
- Add comments and replies
- Tag documents
- Maintain audit logs

---

## ⚙️ Tech Stack
- Python
- Django
- Django REST Framework
- SQLite (for development)
